# Atlas Design System

Welcome to **Atlas** — the shared design system for [TickerTape News](https://tickertapenews.io) and The Desk.

## Structure

- `src/components/` — React components
- `src/tokens/` — Design tokens (colors, typography, spacing, etc.)
- `public/flags/` — SVG country flag icons

## Flags

271 SVG flags available in two aspect ratios, named by ISO 3166-1 alpha-2 country code (lowercase).

| Format | Path | Use case |
|--------|------|----------|
| 4:3 | `/flags/4x3/{code}.svg` | Standard flag shape |
| 1:1 | `/flags/1x1/{code}.svg` | Avatars, icons, circular crops |

### Usage from The Desk / tickertapenews.io

```html
<!-- Swedish flag, 4:3 -->
<img src="https://tickertapenews.github.io/atlas/flags/4x3/se.svg" alt="SE" width="24">

<!-- US flag, square -->
<img src="https://tickertapenews.github.io/atlas/flags/1x1/us.svg" alt="US" width="20">
```

## Font

The design system uses **Figtree** (Google Fonts), loaded via:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Figtree:ital,wght@0,300..900;1,300..900&display=swap" rel="stylesheet">
```
