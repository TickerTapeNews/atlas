# Atlas Design System

The shared design system for **Ticker Tape News** and **The Desk**.

Atlas centralizes components, design tokens, and assets so both projects stay visually consistent.

---

### What's inside

| Section | Description |
|---------|-------------|
| **Colors** | Brand palette and surface colors |
| **Typography** | Figtree font family and scale |
| **Flags** | 271 SVG country flags in 4:3 and 1:1 |

---

### Quick start

Install Figtree and reference Atlas tokens:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Figtree:ital,wght@0,300..900;1,300..900&display=swap" rel="stylesheet">
```

Flag usage:

```html
<img src="https://tickertapenews.github.io/atlas/flags/4x3/se.svg" alt="SE" width="24">
```
