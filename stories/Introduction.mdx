# Atlas Design System

Welcome to **Atlas** — the shared design system for [TickerTape News](https://tickertapenews.io) and The Desk.

## Getting started

Add components, tokens (colors, typography), and stories to build out the design system.

### Structure

- `src/components/` — React components
- `src/tokens/` — Design tokens (colors, typography, spacing, etc.)
- `stories/` — Storybook stories
