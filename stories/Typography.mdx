# Typography

Atlas uses **Figtree** — a variable Google Font supporting weights 300–900 with italic.

export const Sample = ({ weight, size, label }) => (
  <div style={{
    fontFamily: '"Figtree", sans-serif',
    fontWeight: weight,
    fontSize: size || '24px',
    padding: '12px 0',
    borderBottom: '1px solid rgba(15,15,15,0.06)',
    display: 'flex',
    alignItems: 'baseline',
    gap: '16px',
  }}>
    <span style={{ minWidth: '60px', fontSize: '13px', color: 'rgba(15,15,15,0.4)', fontWeight: 400 }}>{label}</span>
    <span>The quick brown fox jumps over the lazy dog</span>
  </div>
);

### Load font

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Figtree:ital,wght@0,300..900;1,300..900&display=swap" rel="stylesheet">
```

```css
body {
  font-family: 'Figtree', sans-serif;
}
```
