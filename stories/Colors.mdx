### Usage

```js
import { colors } from '@atlas/tokens';

// colors.yellow   → '#F6FF00'
// colors.purple   → '#9500FF'
// colors.blue     → '#00AAFF'
```

```css
:root {
  --yellow: #F6FF00;
  --purple: #9500FF;
  --blue: #00AAFF;
  --bg-surface: #F4F4F4;
  --bg-surface-dark: #333333;
  --text-primary: #0F0F0F;
  --text-light: #FAFAFA;
}
```
