### Formats

| Style | Count | Path |
|-------|-------|------|
| Regular (outline) | 1,383 | `/icons/regular/{name}.svg` |
| Solid (filled) | 288 | `/icons/solid/{name}.svg` |

### Usage

```html
<!-- Regular -->
<img src="https://tickertapenews.github.io/atlas/icons/regular/search.svg" width="24" height="24">

<!-- Solid -->
<img src="https://tickertapenews.github.io/atlas/icons/solid/heart.svg" width="24" height="24">
```

### Browse all icons

See the full catalogue at [iconoir.com](https://iconoir.com). All icons are hosted at `/icons/regular/{name}.svg` using the same filenames.
