# Flags

### Formats

| Aspect | Path | Use case |
|--------|------|----------|
| 4:3 | `/flags/4x3/{code}.svg` | Standard flag |
| 1:1 | `/flags/1x1/{code}.svg` | Icons, avatars |

### Usage

```html
<img src="https://tickertapenews.github.io/atlas/flags/4x3/se.svg" alt="SE" width="24">
```
